# General Api

build a task management api with user authentication, task crud operations, and postgresql database

## Tech Stack

- **Backend**: fastapi (python)
- **Frontend**: react (typescript)
- **Database**: postgresql

## Features

### Data Entities

- UserService
- UserRole
- TaskStatus
- TaskPriority
- User
- Task
- UserRegister
- UserLogin
- UserResponse
- Token
- TokenData
- TaskCreate
- TaskUpdate
- TaskResponse

### API Endpoints

- 3 REST API endpoints
